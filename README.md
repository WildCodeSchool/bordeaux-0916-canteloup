* [NodeJS](https://nodejs.org/en/)
* [MongoDB](https://www.mongodb.com/)
* [Brunch](http://brunch.io/)
* [Bower](https://bower.io/)

### Installation

```bash
git clone https://github.com/WildCodeSchool/bordeaux-0916-canteloup.git   
cd bordeaux-0916-canteloup
npm i
```
